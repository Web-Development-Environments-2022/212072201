  Page location Information collecting Technology updates
Web1 The pages are located in portals. User can click on links posted on the portal and arrive the page. The pages are static website. Impossible to add content, or reply or update content without updating the HTML page by owner. The information can be understood from the web is the information written by owner of website. The main new technologies are HTTP, HTML, CSS, URL.
Web2 There are search engines. User can type query and get results. Each result leads to a page. Dynamic website. Viewers can create and share their own content. Based on client-server. Many social networks, blogs. The information can be understood from the web is the information written by owner of website & the information written by users. The main new technology is Client-Server
Web3 There are crawlers scanning the map of the web, and than creating the DB for the search engines. The information can be understood from the web is the information can be understood by AI programs that scan the information written by the users&owners. Many new technologies such as AI, crawler, cloud.

  Web1 Web2 Web3
Searching the "sports" query In web1 we will get a list of portals with the subject of sports. The content will be classified by the owners as related to sports. In web2 we will get a list of sport-related pages, not necessarily with title of "sports" but related to it (by SEO tags for example). In web3 we will get more accurate sport-related pages list, created by a combination of public and private information. The list will be created by AI programs that will make each user the most accurate list for him.
